KalaServer Library

Back home

KalaServer is a lightweight C++ 20 library for Windows. It is aimed for making web and media servers that are hosted locally. At the current stage it works only with a cloudflare tunnel but in the near future it will have a new option to work fully locally via a custom DNS solution so cloudflared etc can only be used as domain registrars.
