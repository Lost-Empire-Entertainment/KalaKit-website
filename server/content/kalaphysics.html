KalaPhysics is a lightweight C++ 20 library for Windows and Linux. It is aimed for game development and comes with a set of primitive physics mechanics, a good gameobject handle system and a good rigidbody component class. At the current stage a lot is still messy because physics is one of the hardest topics for me to learn and understand fully so this library should only be used for experimental projects.
