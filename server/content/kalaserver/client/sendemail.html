⬅ Back
Processes a contact form submission and sends it as an email to the host.
Requires a POST request to the configured route.
method – The request method, should be "POST".
request – Full body content of the request.
clientSocket – Used to send validation or error responses.
clientIP – IP of the client submitting the form.
cleanRoute – Should match the expected email endpoint.
foundRoute – Route definition object for validation.
Used by /client-sent-email
SendEmail(string method, const string& request, uintptr_t clientSocket, string clientIP, string cleanRoute, Route& foundRoute);
