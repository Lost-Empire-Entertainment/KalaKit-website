ConnectionStart()

⬅ Back
Begins processing a new client connection and extracts request info.
Also detects method type, routes, host status, and upload/download intent.
method – Output: HTTP method (GET or POST).
request – Output: full HTTP request string.
clientSocket – Socket used to send/receive data.
clientIP – Output: IP address of the client.
route – Output: full route with query string.
cleanRoute – Output: route without query parameters.
isHost – Output: true if the client is the host machine.
wantsToDownload – Output: true if `?download` is in the URL.
wantsToUpload – Output: true if the method is POST.
Starts connection session
ConnectionStart(string& method, string& request, uintptr_t& clientSocket, string& clientIP, string& route, string& cleanRoute, bool& isHost, bool& wantsToDownload, bool& wantsToUpload);
